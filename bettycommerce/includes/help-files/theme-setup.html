Installation - Top

Chances are if you are reading this you have already installed the theme. In the instance that you are reading this in a text editor than let's break down the available options for installation.

  1. Via FTP:
    1. Unzip the download from MOJO-Themes
    2. Open the folder labeled Theme
    3. Highlight all the files within the Theme folder and move those into the WordPress installation on your server.
    4. wp-content -> themes -> Place Modest Theme files here.
  2. Via WordPress:
    1. Navigate to Appearance -> Themes.
    2. Click Install Themes -> Upload.
    3. Make sure and un-zip the folder you receive from MOJO-Themes. There are many files within this and you will get an error if you just upload your download from MOJO Themes.
    4. Click to Activate the Modest Theme after it has been installed.

Setup - Top

  1. Create Your Pages. Be sure to pick the page template that you want for that page.
    • Default Template: This is where your homepage will be. This pulls your posts.
    • Archives: This template displays the months of all your posts. Enables your user to pick posts according to the month they were written.
    • Contact: This is just like a full-width page. We recommend using Gravity Forms or whatever form plugin of your choice.
    • Full Width - No Sidebar: This is a full width static page.
    • Page With Sidebar: This is a static page with the sidebar.
  2. Go to Appearance -> Menus. Create a menu and select the menu in the "Theme Locations" box. Place the pages you want in your nav and save.
    • Note: It is recommended to not use more than 5 links in your nav.
    • Note: Modest uses only one level deep drop-downs.
  3. Go to Appearance -> Widgets. Select the widgets you would like in your sidebar and in your footer. Press Save.

Modest Theme Options - Top

The Theme Options can be found at the very bottom left of your WordPress.

  • General:
    • Change your Logo: In order to change your logo you need to upload it via the Media tab. Obtain the URL and insert the URL of yoru logo.
    • Twitter User Name: Simply input just your Twitter User Name here.
    • Custom CSS: If you want to override the theme styles simply place valid CSS in this box.
  • Footer:
    • Footer Copyright Text: Simply write whatever you want to display in the footer underneath the widgets.
    • Google Analytics Code: Grab your given code from Google and place in this box.
    • Custom Favicon: Create a Favicon for your site and place the link to the image in this box. You can upload the image in the Media tab.
